# Screenshots

`demo_grid.png` is declared by `packages/drip_flutter/pubspec.yaml` for pub.dev
package screenshots.

The current files are placeholders. Before publishing `drip_flutter`, run
`apps/demo_grid` in profile mode on a device or simulator, capture the 1000-cell
grid, and replace this `demo_grid.png` plus
`packages/drip_flutter/screenshots/demo_grid.png` with that screenshot.
